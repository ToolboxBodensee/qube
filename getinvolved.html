Get involved!

Whether you are a developer, translator or just a normal user, there are many ways to contribute to Qube.

As a developer

Do you want to help developing the Qube-System? Our complete project is and will be completly open source so you can just start developing, and later commit your changes to the Github repository (we need to check it of course). You can either help developing the IDE using HTML, Javascript and PHP or the compiler (which is not ready yet) using QT and C++. If you are familiar with embedded development you can help developing the firmware for our different Qubes or help us by developing the Bootloader of the Chip (both using C++)!

Bug reports

You want to help but you are no developer? If you want to have a new feature or if you want to report a bug you can submit feature requests using the ticket system of Github. Just report the issue, if possible with logs and instructions how to reproduce the issue, this description will make it easier for us to find the issue and resolve it.

Translations

Another method of contributing is by helping to translate the IDE, when we start our beta realeses we will upload everything to a crowdsourced translation service (we are not sure which one) where you can help to translate the IDE to a language you're familiar with.

Donate via Paypal

To cover the costs we have with developing, testing and producing the Qubes it would be nice to support us by donating money to the toolbox Bodensee, our local makerspace!
If you have access to stuff we may need (for example PCB-etching stuff or a server) you can contact us via email (have a look at the contact page) and we can discuss how you can help us best!
