Get involved!

Whether you are a developer, translator or just a normal user, there are many ways to contribute to Qube.

As a developer

Do you want to help developing the Qube-System? Our complete project is and will be completly open source so you can just start developing, and later commit your changes to the Github repository (we need to check it of course).

As a regular user

You want to help but you are no developer? If you want to have a new feature or if you want to report a bug you can submit feature requests using the ticket system of Github. Another method of contributing is by helping to translate the IDE, when we start our beta realeses we will upload everything to a crowdsourced translation service (we are not sure which one) where you can help to translate the IDE to a language you're familiar with.
