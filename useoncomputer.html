Use the IDE on your computer (with a local webserver)

This tutorial will show you how to install the Qube-IDE on your local machine, we first will show you how to install the Apache-Webserver (using the XAMPP-Project) and the how to set up the necessary files. This Method is only for people that have some experience with web-development and don't mind that they maybe get some unreported issues. This method is good for trying the bleeding edge of the development but unsupported!

Only try this if you are an experienced user who wants to try the latest (and maybe unstable) builds!

  • First you need to install XAMPP, this is different on every OS, but a detailed description can be found on the internet
  • Clone or download the Qube repository from Github . Copy the IDE folder in your XAMPP htdocs directory, the exact location depends on your installation, on windows it is C:/xampp/htdocs, on linux /opt/lampp/htcdocs by default
  • Start XAMPP, this can either be done via the XAMPP-Controll Panel on Windows or via the XAMPP-Gui on Linux or via the Terminal (in the default configuration run sudo /opt/lampp/lampp start), you need to run at least the Apache-Webserver and MySQL
  • Open your favourite browser and visit the Website localhost/phpmyadmin
  • In the sidebar select "New" and add a new database it should be called "qubeide".
  • Select the new database in the sidebar, now click on import in the topbar. Select the SQL-Dump which can be found in your local copy of the repository.
    Select UTF-8 as the charset.
    Set the checkbox at "Allow the interruption of an import.",
    write 0 in the next field ("Skip this number of queries"),
    select SQL as format.
    Set the compatibility mode to none and set the checkbox "Do not use AUTO_INCREMENT".
    Now click on "Go" and you should see some new tables in the Database!
